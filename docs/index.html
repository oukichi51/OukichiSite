このブログについて

このブログでは、プログラミング言語・フレームワーク・ツールの使い方などを備忘録として残しています。 Tailwind CSS のユーティリティクラスを活用しているため、HTMLを書くだけで柔軟にデザインを調整できます。

今後、Node.jsやGitHub Actionsによる自動ビルド、Markdown対応なども検討していますが、まずはシンプルに「HTMLファイルとCSSファイルのみ」で運用しています。
